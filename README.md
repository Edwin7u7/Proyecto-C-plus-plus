# Proyecto-C-plus-plus
Pensamiento computacional orientado a objetos 2025
A00842140 Edwin Arturo Salgado Santana

Un cine independiente que apenas esta en su fase de apertura necesita un sistema sencillo para que un usuario pueda consultar las películas disponibles, sus funciones y seleccionar un horario para asistir.
*****Instrucciones de compilación y ejecución*****
- Tener instalado un compilador de C++ (por ejemplo, g++ de MinGW en Windows)
- Tener un IDE como Visual Studio Code, CLion o Dev-C++ para facilitar la edición y ejecución.
- Para compilar el programa en la terminal se debe posicionar en el path de la carpeta donde se encuentra el proyecto.
- Si se compila en Windows se debe utilizar el siguiente comando "g++ main.cpp -o programa"
- Y posteriormente ejecutar el archivo .exe ("programa.exe")

*****Los objetivos del programa son:****
1-Mostrar el catálogo de películas disponibles.

2-Mostrar las funciones (horarios) de cada película.

3-Permitir al usuario elegir:
-Película
-Horario
-Sala
-Asiento

4-Mostrar un resumen final del boleto seleccionado.

****Interacciòn con el usuario******
El usuario podrá:
1-Ver la cartelera (todas las películas disponibles).

2-Ver las funciones (horarios) de una película.

3-Elegir una función específica.

4-Elegir una sala asignada a esa función.

5-Seleccionar un asiento disponible.

6-Confirmar su boleto, mostrando información como:

-Nombre de la película

-Horario

-Sala

-Asiento

-Identificador de boleto
